### not, and, or

| A	    | not A  |
|-------|--------|
| True  | 	False |

#### Таблица истинности для `and`

| A	    | B	     | A and B |
|-------|--------|---------|
| True  | 	True  | 	True   |
| True  | 	False | 	False  |
| False | 	True  | 	False  |
| False | 	False | 	False  |


#### Таблица истинности для `or`

| False | 	False | 	False |
